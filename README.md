# Proyecto Sprint 7 DAG

Proyecto Sprint 7 – TripleTen

## Descripción

Esta aplicación web utiliza **Streamlit**, **Pandas** y **Plotly Express** para explorar un conjunto de datos de anuncios de vehículos contenidos en el archivo `vehicles_us.csv`.  
Su principal funcionalidad es permitir al usuario visualizar un **histograma interactivo** de la columna `odometer` (kilometraje de los autos), lo cual facilita analizar la distribución de recorridos entre los automóviles en venta.

El usuario puede:

- Presionar un **botón** para generar el histograma automáticamente.  
- O bien activar una **casilla de verificación (checkbox)** que también muestra la intención de construir un histograma.
